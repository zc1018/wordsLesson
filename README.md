# 100天核心5000词移动端 Demo

这是根据《100天核心5000词移动端 Demo 开发计划》实现的独立 H5 demo。

## 打开方式

直接用浏览器打开：

`/Users/xdf/Documents/XDF/OPE/100day-core5000-demo/index.html`

无需安装依赖，无需启动服务。

## 当前已实现

- 营前入门测入口：进入课程前先展示核心词汇底盘测评。
- A 段快速识别：模拟分层抽样词和校准项，记录认识/不确定/不认识。
- B 段含义校准：用 4 选 1 题校准真实词义掌握，并识别虚熟词。
- 起点诊断报告：展示 1K-5K 掌握率、熟悉度分布、虚熟词和 5000 词初始标记样例。
- 个性化路径承接：报告页可进入学习样例，学习页显示入门测路径提示。
- 今日学习页：展示 Day 2 词源学和 echo 学习入口。
- 单词学习页：展示 echo、音标、释义、发音反馈、老师讲解占位卡。
- 翻页交互：单词页支持点击提示或左滑翻页进入词源解释页。
- 词源解释页：顶部 banner 已接入 `assets/echo-etymology-image2.mp4`，翻页后自动播放，并展示 Greek -> Latin -> English 的词源路径。
- 互动练习页：实现单选题和词源匹配题。
- 反馈页：练习完成后展示 echo 已掌握和同课扩展词。

## 说明

老师讲解视频按当前要求暂留空，仅保留可点击占位和反馈提示。词源故事视频已接入本地 MP4。

Demo 使用静态 HTML/CSS/JS 实现，后续可迁移为 React/Vite 项目或嵌入正式学习系统。
